# Usage Examples

## Example: Adding a New Feature Card

In `features.html`:
```html
<div class="feature">
  <h4>Custom Skins</h4>
  <p>Upload and use your own Minecraft skin in-game.</p>
</div>
```

## Example: Adding a New Section

In `index.html`:
```html
<section id="faq">
  <h3>FAQ</h3>
  <p>Frequently asked questions go here.</p>
</section>
```
